# netflix-python-project
## Netflix Movies Data Analysis

This project performs exploratory data analysis on a dataset of 9,800+ Netflix movies.

### Tools
- Python
- Pandas
- Matplotlib
- Seaborn

### Key Analysis
- Genre distribution and popularity trends
- Vote average categorization using feature engineering
- Year-wise release analysis

### Outcome
Derived insights on most frequent genres, popularity patterns, and release trends.
